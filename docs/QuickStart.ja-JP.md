# Vex クイックスタート

## 作成または開く

- **ファイル > 新規** または `Ctrl+N` で新しい Markdown 文書を作成します。
- **ファイル > 開く** または `Ctrl+O` で単一ファイルを開きます。
- **ファイル > フォルダーを開く** で文書フォルダーを左側のファイル一覧に読み込みます。

## 書く、移動する

- 中央のエディターで Markdown を編集すると、右側のプレビューが更新されます。
- **アウトライン** タブを使うと見出し間をすばやく移動できます。
- ソース編集に集中したい場合は **表示 > ソースモード** を使います。

## 検索、置換、整形

- `Ctrl+F` で検索、`Ctrl+H` で置換を開きます。
- `Aa`、`W`、`.*` はそれぞれ大文字小文字の区別、単語単位、正規表現検索を表します。
- 段落メニュー、書式メニュー、またはエディターのコンテキストメニューから、見出し、リスト、表、リンク、画像、一般的な Markdown 書式を挿入できます。

## エクスポートと印刷

- **ファイル > エクスポート > HTML** で単独の HTML 文書を生成します。
- **ファイル > エクスポート > PNG** で縦長画像を書き出します。
- **ファイル > エクスポート > PDF** でページ分割された PDF を生成します。
- **ファイル > エクスポート > Word** で Markdown 構造とローカル画像を含む `.docx` 文書を生成します。
- **ファイル > 印刷** でシステムブラウザーの HTML 印刷プレビューを開きます。
- エクスポート成功後、対応プラットフォームでは保存先フォルダーを開き、出力ファイルを選択します。

## 設定

- テーマ、タイポグラフィ、コンパクトレイアウト、言語、サイドバー、プレビュー、ステータスバー、常に手前に表示、エディター倍率、行番号の設定はローカルに保存されます。
- 初回ガイドは **ヘルプ > 初回ガイド** から再度開けます。
