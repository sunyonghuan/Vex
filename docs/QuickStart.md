# Vex 快速开始

Vex 是一个基于 Avalonia 12 的 Markdown 编辑器，产品中文名为“维刻”，Slogan 是“极简之力，妙笔成章”。

## 基础操作

1. 通过“文件 / 新建”创建空白 Markdown 文档。
2. 通过“文件 / 打开”打开单个 Markdown 文件。
3. 通过“文件 / 打开文件夹”加载文件列表；直接打开单个文件时，也会自动加载同目录 Markdown 文件。
4. 在中间编辑区输入 Markdown，右侧会显示预览。
5. 从网页复制内容后粘贴到中间编辑区，Vex 会优先读取剪贴板 HTML 并转为 Markdown；普通文本粘贴仍会照常回落。
6. 通过“帮助 / 主题色”和“帮助 / 排版”切换应用主题色、排版主题和紧凑布局。
7. 通过“文件 / 导出”导出 HTML、PDF、PNG 或 Word（.docx）；PDF、PNG 和 Word 会处理本地、`data:image`、HTTP(S)、SVG/GIF/WebP 图片，PDF 正文可选择和复制，PDF 和 Word 也会把图片嵌入文件，离线发送后仍可查看。
8. 通过“文件 / 复制到公众号”“复制到知乎”“复制到稀土掘金”把当前 Markdown 写入富 HTML 剪贴板；Vex 会把当前 Markdown、排版主题和发布目标交给 `CodeWF.Markdown` 公共复制接口，粘贴到对应网页编辑器时会按当前排版主题渲染。
9. 使用 `Ctrl+F` 查找、`Ctrl+H` 替换；查找和替换输入框限制为单行，避免超长文本撑乱布局。

## 当前版本

1.1.0 是当前版本，已具备产品骨架、菜单结构、文档读写、Markdown 预览、同目录文件列表、常用导出、主题持久化和关键 UrsaWindow 对话框。
