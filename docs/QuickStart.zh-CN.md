# Vex 快速开始

Vex 是一个基于 Avalonia 12 的 Markdown 编辑器，产品中文名为“维刻”，Slogan 是“极简之力，妙笔成章”。

## 基础操作

1. 通过“文件 / 新建”创建空白 Markdown 文档。
2. 通过“文件 / 打开”打开单个 Markdown 文件。
3. 通过“文件 / 打开文件夹”加载文件列表；直接打开单个文件时，也会自动加载同目录 Markdown 文件。
4. 在中间编辑区输入 Markdown，右侧会显示预览。
5. 通过“帮助 / 主题色”和“帮助 / 排版”切换应用主题色、排版主题和紧凑布局。
6. 通过“文件 / 导出”导出 HTML、PDF、PNG 或 Word（.docx）；导出成功后会打开保存目录并定位文件。
7. 使用 `Ctrl+F` 查找、`Ctrl+H` 替换；查找和替换输入框限制为单行，避免超长文本撑乱布局。

## 当前版本

0.1.0 是基础开发版本，已具备产品骨架、菜单结构、文档读写、Markdown 预览、同目录文件列表、常用导出、主题持久化和关键 UrsaWindow 对话框。
