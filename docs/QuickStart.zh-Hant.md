# Vex 快速開始

## 建立或開啟

- 使用 **檔案 > 新增** 或 `Ctrl+N` 建立新的 Markdown 文件。
- 使用 **檔案 > 開啟** 或 `Ctrl+O` 開啟單一檔案。
- 使用 **檔案 > 開啟資料夾** 將文件資料夾載入左側檔案清單。

## 撰寫與導覽

- 在中間編輯器撰寫 Markdown，右側預覽會同步更新。
- 使用 **大綱** 頁籤在標題之間快速跳轉。
- 需要專注編輯原始碼時，可使用 **檢視 > 原始碼模式**。

## 尋找、取代與格式

- 使用 `Ctrl+F` 尋找，使用 `Ctrl+H` 取代。
- `Aa`、`W` 和 `.*` 選項分別對應區分大小寫、整詞比對和正則搜尋。
- 段落與格式選單、或編輯器右鍵選單，可插入標題、清單、表格、連結、圖片和常用 Markdown 樣式。

## 匯出與列印

- 使用 **檔案 > 匯出 > HTML** 產生獨立 HTML 文件。
- 使用 **檔案 > 匯出 > PNG** 產生長圖，並解析本地、`data:image`、HTTP(S)、SVG、GIF 與 WebP 圖片。
- 使用 **檔案 > 匯出 > PDF** 產生分頁 PDF，圖片會嵌入檔案，離線分享後仍可查看。
- 使用 **檔案 > 匯出 > Word** 產生 `.docx` 文件，保留 Markdown 結構並嵌入圖片，必要時會將 SVG/GIF/WebP 轉為 PNG。
- 使用 **檔案 > 列印** 在系統瀏覽器中開啟 HTML 列印預覽。
- 使用 **檔案 > 複製到公眾號**、**複製到知乎** 或 **複製到稀土掘金**，可將目前 Markdown 以套用目前排版主題的富 HTML 寫入剪貼簿。
- 匯出成功後，Vex 會在平台支援時開啟儲存位置並選取匯出的檔案。

## 設定

- 主題、排版、緊湊版面、語言、側邊欄、預覽、狀態列、視窗置頂、編輯器縮放與行號設定會儲存在本機。
- 首次啟動的新手引導可從 **說明 > 新手引導** 再次開啟。
