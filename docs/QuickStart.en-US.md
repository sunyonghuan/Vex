# Vex Quick Start

## Create Or Open

- Use **File > New** or `Ctrl+N` to start a new Markdown document.
- Use **File > Open** or `Ctrl+O` to open one file.
- Use **File > Open Folder** to load a document folder into the left file list.

## Write And Navigate

- Edit Markdown in the center editor; the preview updates on the right.
- Use the **Outline** tab to jump between headings.
- Use **View > Source Mode** when you want a focused source-editing layout.

## Find, Replace, And Format

- Use `Ctrl+F` for Find and `Ctrl+H` for Replace.
- Use the `Aa`, `W`, and `.*` options for match case, whole word, and regular expression search.
- Use the Paragraph and Format menus, or the editor context menu, for headings, lists, tables, links, images, and common Markdown formatting.

## Export And Print

- Use **File > Export > HTML** for a standalone HTML document.
- Use **File > Export > PNG** for a long image export.
- Use **File > Export > PDF** for a paged PDF export.
- Use **File > Print** to open the HTML print preview in the system browser.

## Settings

- Theme, typography, compact layout, language, sidebar, preview, status bar, editor zoom, and line-number settings are saved locally.
- The first-run guide can be opened again from **Help > New User Guide**.
